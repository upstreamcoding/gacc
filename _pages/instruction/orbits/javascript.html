Overview

Become competent in JavaScript by building web apps and integrating JSON data.

You'll build a portfolio of web apps from scratch using JavaScript.

The member-instructor ratio is 5:1 to provide a personalized learning experience.


Curriculum

  • JS Primitives & Programming Fundamentals
    Create a strong JavaScript foundation by building a directory application leveraging Functions, Objects and Operators.
  • Clean Code
    Fine-tune your JS skills through refactoring and begin manipulating the DOM while building a quiz application using Control Flow principles.
  • Bring on the Data
    You will build an application that pulls data from other servers to create an exciting API application that consumes JSON data.
  • Application Design & ES6+
    Begin using the latest ES6+ syntax including arrow functions, destructuring, Modules, template literals & interpolation while building an HR application tracks the relationship between managers and employees.
