Overview

Become competent in the main concepts of building web apps with React.

You'll build a portfolio of web apps from scratch using React.

The member-instructor ratio is 5:1 to provide a personalized learning experience.


Curriculum

  • JSX, State & JS (ES6+)
    Start thinking in React, master JavaScript ES6+ builtin functions and write & refactor HTML into JSX syntax.
  • Local Development, Components & Props
    Develop locally using Node. Break down User Interfaces into small reusable peices. Build functional components written using JSX syntax that receive JSON data through props.
  • Composition, Conditional Rendering & Deployment
    Compose Class components that conditionaly render elements that work in-concert with each other via component-level state. Render lists by maping data and returning HTML elements. Deploy your React Apps to production.
  • Event Handling, Component Lifecycle & APIs
    Consume data from APIs. Connect components together via a centralized state. Handle component-level events and lift state up to pass events from children up to the parent component.
